DnD5e Alcohol — FoundryVTT Module

A FoundryVTT module that adds a catalogue of alcoholic beverages, items, monsters, feats etc, complete with mechanics and data bindings.

Repository↗FoundryVTT Module Page↗

DnD5e Alcohol is a module for the TTRPG software FoundryVTT that adds alcoholic beverages to the game.

Working on this project was my first foray into a larger JavaScript ecosystem…

The module is based on a great, free ruleset by CupidFaust on GMBinder that I converted with their blessing.
